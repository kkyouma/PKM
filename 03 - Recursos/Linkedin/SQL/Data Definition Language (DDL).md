# Data Definition Language (DDL)

Se conocen como **DDL** los que definen la estructura de nuestra [[Base de datos|BD]]. Algunos comandos son:

- `CREATE`
- `ALTER`
- `DROP`
- `TRUNCATE`
- `RENAME`
- `COMMENT`
